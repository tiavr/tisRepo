L'évolution des OS



Maintenant que vous en savez plus sur les systèmes d'exploitation, venez en découvrir plus sur les OS les plus connus !




  • Windows

Windows




De nos jours, Windows est le système d'exploitation le plus utilisé sur ordinateur. En effet, il est déjà pré-installé sur une grande majorité d'ordinateurs, ce qui contribue à sa popularité. Il existe aussi une version pour les serveurs.




1985 : Microsoft lance Windows 1, la toute première version de Windows; il s’agissait plutôt d’une interface.
1987 : Windows 2; ajouts de fonctionnements pour les fenêtres comme agrandissement et réduction ainsi que la superposition.
1990 : Windows 3; évolution du design, Windows prend de l’ampleur et devient un concurrent de Macintosh.
1995 : Windows 95; se rapproche de la version de Windows que l’on connaît aujourd’hui avec l’organisation du bureau avec la barre de tâche par exemple.
1998 : Windows 98; Internet arrive enfin !
2000 : Windows 2000; la version Professional s’impose auprès des entreprises.
2001 : Windows XP; la version est très appréciée du grand public grâce à ses couleurs plus vives et un design plus élaboré.
2007 : Windows Vista; système d’exploitation peu stable et très lent mais avec l’innovation de la transparence en terme de design.
2009 : Windows 7; moins lent que la précédente version et plus stable.
2012 : Windows 8; menu se présente comme sur les versions mobiles et tablettes, avec des “tuiles”. Changement un peu trop anticipateur.
2015 : Windows 10; dernière version en date, reprise des meilleurs points des deux dernières versions. L’interface se présente de façon beaucoup plus minimaliste que précédemment.






Android


Actuellement Android est le système d'exploitation le plus répandu sur les smartphones.




2003 : naissance de la société Android.
2005 : rachat par Google.
2007 : découverte par le monde d’Android.
2008 : Android 1.0; sur le téléphone HTC dream. Il comporte des caractéristiques basiques comme l’envoi de sms, l’appareil photo … mais aussi Youtube, la possibilité de faire des recherches sur internet avec Google, et bien d’autres encore.
2009 : Android 1.5 Cupcake; Android appelle dorénavant ses versions suivant des gâteaux. Widgets dorénavant disponible ainsi que le copier/coller dans le navigateur.
          Android 1.6 Donut; possibilité de sélection multiple pour la suppression dans la galerie et recherche vocale.
          Android 2.0 Eclair; amélioration du clavier tactile et apparition de Google Maps.
2010 : Android 2.2 Froyo; mot de passe pour l’écran de verrouillage, Google Play remplace Android Market.
          Android 2.3 GingerBread; disponibilité d’un gestionnaire de téléchargement.
2011 : Android 3.0 HoneyComb; réservée pour les tablettes graphiques.
          Android 4.0 Ice Cream Sandwich; beaucoup de nouveautés comme la capture d’écran, la possibilité de pouvoir connaître sa consommation internet ou encore la présence de dictionnaires dans les claviers. Ainsi que la possibilité de créer des dossiers plus facilement.
2012 : Android 4.1 Jelly Bean; Google Chrome devient le navigateur par défaut, un écran de veille est aussi ajouté.
2013 : Android 4.4 KitKat; l’application de messagerie Hangouts fait sa place. Ajout d’émojis dans le clavier.
2014 : Android 5.0 Lollipop; amélioration pour le système des notifications, projet sur la consommation de batterie visant à la réduire ainsi que gain en autonomie.
2015 : Android 6.0 Marshmallow; mode multi-fenêtres disponible.
2016 : Android 7.0 Nougat; mode nuit disponible, présence d’un bouton 'Tout effacer' dans le multi-tâches.
2017 : Android 8.0 Oreo; possibilité de regarder des films sur Google Play, limite des activités en arrière-plan.
2018 : Android 9.0 Pie; nouvelles fonctionnalités comme l'optimisation intelligente de la batterie ou de la luminosité. Filtre anti-lumière bleue.






Mac-OS et iOS


Les systèmes Mac Os et iOS sont des systèmes d'exploitation développés par Apple, spécifiquement pour les produits Apple.



1984 : System; premier système d’exploitation de Apple.
1997 : devient Mac OS 8; interface graphique nouvelle.
2001 : Mac OS X 10.1; système plus rapide que les précédents.
2005 : Mac OS X 10.4; système encore plus rapide, présence du système de recherches Spotlight.
2012 : Mac OS X 10.8; possibilité d’accès aux réseaux sociaux intégré, l’application Ichat est remplacée par Messages.
2014 : Mac OS X 10.10; le design évolue beaucoup vers le “flat design” avec un mode nuit en plus.
2016 : Mac OS X 10.12; Apple Pay et Siri disponibles.
2019 : Mac OS X 10.15; prend en charge exclusivement des applications 64 bits. possibilité de contrôle avec des commandes vocales.






2007 : iPhone OS 1; premier système d’exploitation pour téléphone.
2010 : : iOS 4; iPhone OS devient iOS. Multi-tâches et FaceTime.
2011 : iOS 5; fonctionnalités comme Siri et iMessage.
2015 : iOS 9; amélioration d’applications comme les Notes ou Maps.
2017 : iOS 11; Apple Pay Cash et Face ID.
2018 : iOS 12; FaceTime de groupe, nouvelle application pour mesurer : Mesure.






Linux

1991 : Linus Torvalds crée le noyau Linux, qui combiné avec GNU va devenir le système d’exploitation GNU/Linux.

Linux a l’avantage de proposer des mises à jour plus fréquemment que les autres OS, ainsi que d’être gratuit. Cet OS est différent des autres de par son système de distribution. En effet, ces distributions sont une sorte de regroupement de logiciels et d’applications qui ont pour objectifs des utilisations différentes.

On peut en citer quelques uns comme Debian, Red Hat, Mandriva ....
Chaque distribution est développée par différents acteurs. Debian est la seule qui n’est pas générée par des entreprises.

La distribution Ubuntu doit être une de celles qui est considérée comme la plus populaire de toutes les distributions. Elle est destinée à être accessible à toute personne. Une nouvelle version est disponible tous les 6 mois.
La dernière version en date, celle de avril 2019 : Ubuntu 19.04, comporte le noyau Linux 5.0 ainsi que l’interface GNOME 3.32.
